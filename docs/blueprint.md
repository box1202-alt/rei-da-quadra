# **App Name**: REI DA QUADRA

## Core Features:

- Admin: Court & Modality Setup: Administrator login (numeric password) to add or configure available footvolley courts and game modalities via a dedicated action button.
- Add Player Pairs to Queue: A prominent '+' button allows any user to easily add new player pairs ('duplas') to the waiting list.
- Live Game Display: Visually presents the current match on a simplified court graphic, clearly showing the two playing pairs.
- Dynamic Waiting List: Displays the ordered list of upcoming player pairs, updating automatically based on game progression and rules.
- Victory & Consecutive Wins Tracking: Includes a dedicated 'victory button' to mark a winning pair and automatically tracks consecutive wins for the team on court.
- Automated '2-Wins' Game Logic: Automatically enforces the '2 consecutive wins' rule, managing pair rotation: pairs with 2 wins leave, two new pairs from the queue enter, winning pair returns to queue position #1, losing pair to the end.

## Style Guidelines:

- Primary color: Energetic Orange (#FF8000) for high-impact titles like 'REI DA QUADRA' and key interactive elements. Chosen for its vibrancy and association with sport and energy, complementing a dark background.
- Background color: Deep Black (#000000), providing a strong, sleek, and focused aesthetic that allows primary and accent colors to pop. Matches user request for a 'preto e laranja' visual.
- Accent color: Pure White (#FFFFFF) used for subtitles like 'BEACH SPORTS PORTAL', crucial text, and iconography to ensure maximum readability and strong contrast against the black background and orange highlights.
- Body and headline font: 'Poppins' (sans-serif) for a modern, athletic, and clean look. This font will be used for all text, including titles, subtitles, and list items, ensuring consistency and readability in a 'simples estilo esportivo'.
- App Icon: Features a clear illustration of a 'bola com uma coroa', representing 'REI DA QUADRA' and the sport. Other in-app icons should maintain a 'visual simples estilo esportivo'.
- Overall layout is 'simples e visual', prioritizing clear hierarchy and intuitive navigation. Game elements like 'Em Quadra' and 'Próximas Duplas' will use distinct sections, with the court display dividing vertically.
- Subtle, fluid animations for queue reorganization and when new teams enter or leave the court, enhancing user feedback without being distracting.
